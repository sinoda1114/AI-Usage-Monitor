# Privacy Policy for AI Usage Monitor

AI Usage Monitor is a Chrome extension that displays usage information from Cursor, Codex, and Claude in a browser popup.

## Information handled by this extension

This extension reads usage metrics displayed on the following service pages:

- Cursor usage / spending pages
- ChatGPT Codex usage analytics page
- Claude usage settings page
- Devin organization usage settings page (`app.devin.ai/org/{slug}/settings/usage`), when configured in extension settings

The extension may process the following information locally:

- Usage metrics displayed on those pages
- Reset times shown on those pages
- Remaining credit or usage limit information shown on those pages
- Page URL and page title, for diagnostic and display purposes

## Local storage

The extracted information is stored only in the user's Chrome browser using Chrome local storage.

The extension also stores local preferences only in the browser, including popup display order, which providers are shown, whether automatic refresh is paused, the automatic refresh interval (in minutes), UI language choice, and your Devin organization slug when you enable Devin collection.

## No external transmission

AI Usage Monitor does not send collected information to any external server.

The extension does not:

- Sell user data
- Share user data with third parties
- Use user data for advertising
- Use user data for analytics
- Collect passwords
- Collect authentication tokens
- Collect payment information

## Purpose of data usage

The information is used only to display AI service usage metrics inside the extension popup.

For support, bug reports, or feature requests, please use GitHub Issues:

# AI Usage Monitor プライバシーポリシー

AI Usage Monitor は、Cursor・Codex・Claude の使用量情報を Chrome のポップアップに表示するための拡張機能です。

## この拡張が扱う情報

この拡張は、以下のサービスの使用量ページに表示される情報を読み取ります。

- Cursor の使用量 / spending ページ
- ChatGPT Codex の usage analytics ページ
- Claude の usage settings ページ
- Devin の組織ごとの使用量ページ（`app.devin.ai/org/{スラッグ}/settings/usage`。拡張の設定で組織スラッグを指定した場合）

この拡張は、以下の情報をブラウザ内で処理する場合があります。

- 使用量メトリクス
- リセット時刻
- 残りクレジットや使用制限に関する情報
- 診断・表示目的のページ URL とページタイトル

## 保存先

取得した情報は、Chrome ブラウザ内のローカルストレージにのみ保存されます。

ポップアップの表示順・表示するプロバイダー・自動更新の一時停止状態・自動更新の間隔（分）・表示言語・Devin を使う場合の組織スラッグなどの設定も、同様にブラウザ内にだけ保存されます。

## 外部送信について

AI Usage Monitor は、取得した情報を外部サーバーへ送信しません。

この拡張は、以下を行いません。

- ユーザーデータの販売
- 第三者への提供
- 広告利用
- 分析利用
- パスワードの取得
- 認証トークンの取得
- 支払い情報の取得

## 利用目的

取得した情報は、拡張機能のポップアップ内に AI サービスの使用量を表示する目的にのみ使用します。
